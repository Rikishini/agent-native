# Assets

Agent-native digital asset libraries for brand-consistent image and video generation.

Use libraries and folders to collect uploaded images, videos, logos, product
shots, diagrams, style briefs, generated candidates, and saved assets. The UI
curates the DAM; the agent handles generation, search, and iteration through
actions and A2A.

Every generated asset writes an audit run with user prompt, compiled prompt,
model, source app, references, outputs, status, and refinement lineage. Org
admins can review the log at `/audit` and export CSV for design-team QA.
